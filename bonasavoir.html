Imprimer avec DOS sur imprimante USB

dans l'onglet "PORT" de la fenetre propriete de l'imprimante, cocher la case "Activer le pool de l'imprimante" et cocher le port de l'primante utilisé par le programme dos  (LPT1,LPT2...)

Cartouche Epson :  différences entre T089x  et T071x

T089x : Economique et qualité correcte, faible contenance (5,8ml)
T071x : Plus chère, qualité meilleure, contenance supérieure (7,4ml)

Ireport : Format Papier 

lorsque vous utilisez un format papier dont la largeur depasse la longueur, ireport inverse automatiquement l'impression en mode Paysage. Pour forcer le mode Portrait, il suffi d'ouvrir le fichier ".JRXML" à l'aide d'un editeur de texte et de modifier la ligne "orientation="Portrait" ".


JAVA : Composante non rafraichi avant un sleep

Java a tendance à grouper plusieurs "paint" avant de les afficher. Pour forcer le rafraichssement immédiat , il faut utiliser "paintImmediately" .
